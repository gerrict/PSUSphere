# PSUSphere

## Short Description
PSUSphere is a web application that helps manage student organizations, programs, and memberships at Palawan State University. It allows admins to create, view, and manage students, organizations, and members efficiently.

## Features
- Add, edit, and view **Colleges** and **Programs**
- Manage **Organizations** with descriptions and associated colleges
- Register **Students** with unique IDs and program enrollment
- Track **Organization Members** and their joining dates
- Admin interface powered by **Django Admin**
- Generate initial test data using a management command
